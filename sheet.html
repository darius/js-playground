Click anywhere on the canvas to create a new point.

On an existing point: click to select or deselect it. Drag it to move it around. (But you can't select/drag the constants 0, 1, i.)

To add: select a point a, then drag from 0 to a point b. The sum a+b appears. (On a touchscreen, you'd do this by panning instead.)

To multiply: select a point a, then drag from 1 to a point b. The product ab appears. (On a touchscreen, you'd do this by pinching instead.)
